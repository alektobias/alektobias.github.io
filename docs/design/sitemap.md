# Portfolio Sitemap

This document outlines the structure of the portfolio website.

## 1. Top-Level Pages
- **Home (`/`)**: The main experience. includes Story, Selected Work, Skills, and Timeline.
- **Projects (`/projects`)**: Full archive of work.
- **Blog (`/blog`)**: Articles and thoughts.

## 2. Dynamic/Nested Routes
- **Project Detail (`/projects/[slug]`)**: In-depth look at a specific project (case study).
- **404 Page**: Custom error page.
- **RSS Feed (`/rss.xml`)**: For blog subscriptions.
- **Sitemap (`/sitemap-index.xml`)**: SEO generated sitemap.
